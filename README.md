 🌱 SoilSense

Overview

SoilSense is a web application that helps users analyze soil conditions and make better agricultural decisions. It provides insights based on soil data and helps improve crop productivity.

---

Features

* Soil data analysis
* User-friendly interface
* Real-time insights
* Responsive UI

---
 Tech Stack

* **Frontend:** React.js, HTML, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Tools:** Git, REST APIs

---

## ▶️ Run Locally

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
node server.js
```
